# Worm Castings Experience — Wildwood Hills Ranch of Iowa

Prototype web experience for worm castings **sales, education, and kids engagement**.

**Live:** https://yoans.github.io/worm-castings/

## Run

```bash
npm install
npm run dev
```

## Routes

| Path | Purpose |
|------|---------|
| `/` | Brand hub |
| `/lawn` | Calculator, before/after, nitrates/water-ban pitch, testimonials, call/visit buy |
| `/learn` | Portable pamphlet — nitrates → castings |
| `/play` | Kids worm-farm game (5–10 + parent) |
